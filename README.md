## I build tools that make complicated work feel simple.

I'm a software engineer in Vancouver focused on full-stack product engineering, AI workflows, and clean user-facing systems. I like taking messy operational workflows, designing the interface around the real user behavior, and then building the backend, data model, and automation needed to make it fast.

Recently, that has meant:

- Building AI-assisted CRM workflows that reduce manual report, email, task, and insight generation time.
- Designing and implementing modern React, TypeScript, Shadcn/UI, and React Query product surfaces.
- Shipping backend systems for high-volume email campaigns, workflow automation, observability, and dashboard infrastructure.
- Turning early product ideas into working systems, from Figma and prototype to production code.

## What I Care About

- Interfaces that are quiet, fast, and hard to misuse.
- AI features that do real workflow work, not just autocomplete text.
- Systems with typed boundaries, useful observability, and clear failure modes.
- Product engineering that starts with the user problem and ends with maintainable code.
